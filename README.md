# 📉 Customer Retention & Churn Analysis
**Future Interns | Data Science & Analytics | Task 2**

## 🔍 Overview
Analysis of Telco customer data to identify churn patterns, 
retention drivers, and customer lifetime trends.

## 📁 Dataset
- **Source:** Telco Customer Churn Dataset (Kaggle)
- **Records:** 7,032 customers | 21 columns

## 🛠️ Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Google Colab | Tableau

## 📈 Key Insights
- **Churn Rate:** 26.6% (1,869 out of 7,032 customers)
- **Highest Risk:** Month-to-month contracts → 42.7% churn rate
- **Internet Service:** Fiber optic customers churn most (~42%)
- **Tenure Impact:** Churned customers stay avg 18 months vs 37.7 months retained
- **Charges:** Churned customers pay higher monthly charges ($74.44 avg)

## 💡 Recommendations
1. Convert month-to-month customers to annual contracts with discounts
2. Investigate Fiber optic service quality issues
3. Focus retention efforts on customers in first 12 months
4. Offer loyalty rewards for customers crossing 24+ months
